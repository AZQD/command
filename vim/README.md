Vim是从 vi 发展出来的一个文本编辑器。
mac中如何从vim文本编辑器退回到命令
有的时候，我们经常从命令行中进入vim文本编辑器:

1.vim进入编辑模式
常用方式：只需按a键即可，表示在当前字符后插入；

也可以使用：

i：在当前字符前插入。

I：在光标所在行的行首插入。

a：在当前字符后插入。

A：在光标所在行的行尾插入。

o：在当前行的下一行插入新的一行。

O：在当前行的上一行插入新的 一行。


2.vim退出编辑模式

点击 Esc 键，Vim 进入命令模式。然后输入：

:q  ——退出（这是 :quit 的缩写）
:q! ——不保存退出（这是  :quit! 的缩写）
:wq ——写入文件并退出；（这是 :writequit 的缩写）
